# KANA
C Near field correlator and python scripts to execute correlation 

## Dependencies
- Cmake
- MPI
- FFTW
- vdifio
- python 3
  - ply
  - matplotlib
 
# Installation
```bash
mkdir build
cd build
cmake ..
make
sudo make install
```
